Cardify es un plugin de jQuery que dado un contenedor debe buscar todas las imágenes que encuentre dentro del contenedor y reemplazarlas por un nuevo elemento <figure> que contenga la imagen <img> además de un <figcaption> con el texto del atributo alt de la imagen.

Instalación de la librería

Para incluir este plugin a tu proyecto, inserta el archivo index.js en una etiqueta script en tu archivo HTML:

<script src="js/index.js"></script>
Deberá tener la siguiente estructura:
  1. librería jQuery:
    2. <script src="vendors/jquery/jquery-3.2.1.min.js"></script>
  2. Framework: Materialize
    3. <script src="vendors/materialize/js/materialize.min.js"></script>
  3. Archivos para el funcionamiento del plugin:
    4. <script src="js/index.js"></script>
    <script src="js/app.js"></script>

Deslice el mouse sobre una imagen y aparecerá la descripción del mismo.
